What is WRSS Builder?

WRSS Builder v2 extends the original Builder with live data visualization, Posit Cloud model execution, and full results analytics. Build basin models visually, run them in the cloud, and explore simulation outputs — all in one tool.

New in v2

• Live Charts — every time series and rating curve input now has a Chart tab for instant visualization.
• Run on Posit Cloud — download the R script, run it in Posit Cloud, export the result JSON, and import it back for full analytics.
• Results Panel — storage, inflow/outflow, spill, evaporation, supply/deficit charts with KPI cards and CSV/JPEG downloads.
